Gamma Distribution#

The Gamma Distribution is a continuous probability distribution used to model the time until the occurrence of a certain number of events in a Poisson process (a process where events happen independently at a constant rate).

It generalizes the Exponential Distribution:

  • Exponential distribution = time until the first event.

  • Gamma distribution = time until the k-th event.

Parameters of the Gamma Distribution#

The Gamma distribution is described by two parameters:

  1. Shape parameter (\(k\) or \(\alpha\)):

    • Represents the number of events we are waiting for.

    • Must be > 0.

    • When \(k = 1\), the Gamma distribution becomes the Exponential distribution.

  2. Rate parameter (\(\lambda\)) OR Scale parameter (\(\theta = 1/\lambda\)):

    • \(\lambda\) is the rate at which events occur.

    • \(\theta\) is the average waiting time between events.

Probability Density Function (PDF)#

For \(x > 0\):

\[ f(x; k, \lambda) = \frac{\lambda^k x^{k-1} e^{-\lambda x}}{\Gamma(k)} \]

where:

  • \(\Gamma(k)\) is the Gamma function, which generalizes factorials:

    \[ \Gamma(k) = (k-1)! \quad \text{if $k$ is an integer.} \]

Statistical Measures#

  • Mean:

    \[ \mu = \frac{k}{\lambda} \]

  • Variance:

    \[ \sigma^2 = \frac{k}{\lambda^2} \]

  • Skewness:

    \[ \text{Skewness} = \frac{2}{\sqrt{k}} \]

  • Kurtosis (excess):

    \[ \text{Kurtosis} = \frac{6}{k} \]

Applications#

  • Waiting times: Time until multiple arrivals in a queue (e.g., time until 5th customer arrives).

  • Rainfall modeling: Total rainfall in a given time period.

  • Insurance/Finance: Modeling claim sizes or risk assessment.

  • Engineering: Time-to-failure models when failure depends on multiple causes.

Example:#

Suppose a call center receives calls at a rate of 2 per minute. What is the distribution of the waiting time until the 3rd call?

  • \(k = 3\) (waiting for 3 events)

  • \(\lambda = 2\) (calls per minute)

Mean waiting time:

\[ \mu = \frac{k}{\lambda} = \frac{3}{2} = 1.5 \ \text{minutes} \]

Variance:

\[ \sigma^2 = \frac{k}{\lambda^2} = \frac{3}{4} = 0.75 \]

So the Gamma(3, 2) distribution models the waiting time for the 3rd call.

import numpy as np
import matplotlib.pyplot as plt
from scipy.stats import gamma

# Parameters for Gamma distribution
k = 3      # shape parameter (number of events)
lam = 2    # rate parameter (events per unit time)
theta = 1/lam  # scale parameter

# Generate random samples from Gamma distribution
samples = gamma.rvs(a=k, scale=theta, size=10000)

# Define x values for PDF curve
x = np.linspace(0, 5, 1000)
pdf = gamma.pdf(x, a=k, scale=theta)

# Plot histogram of samples with PDF curve
plt.figure(figsize=(8,5))
plt.hist(samples, bins=40, density=True, alpha=0.6, color='skyblue', label="Simulated Data (Samples)")
plt.plot(x, pdf, 'r-', lw=2, label=f"Gamma PDF (k={k}, Œª={lam})")
plt.title("Gamma Distribution (Waiting time until 3rd event, Œª=2)")
plt.xlabel("x (waiting time)")
plt.ylabel("Density / Probability")
plt.legend()
plt.grid(alpha=0.3)
plt.show()
