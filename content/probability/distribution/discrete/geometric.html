Geometric Distribution#

The Geometric Distribution models the number of trials needed until the first success occurs in a sequence of independent Bernoulli trials (yes/no experiments).

üîë Key Concepts#

  • Bernoulli Trial: An experiment with exactly two outcomes: success (probability = \(p\)) or failure (probability = \(1-p\)).

  • Geometric Random Variable (X): The number of trials until the first success happens.

For example:

  • Tossing a coin until it lands heads.

  • Testing products until finding the first defective one.

  • Calling customers until the first person answers.

üìù Formulas#

  1. Probability Mass Function (PMF):

\[ P(X = k) = (1-p)^{k-1} \cdot p, \quad k = 1, 2, 3, \dots \]

  • \(p\): Probability of success

  • \(k\): The trial on which the first success occurs

üëâ Example: If \(p = 0.2\), then the probability that the first success happens on the 3rd trial is: \((1-0.2)^{2} \cdot 0.2 = 0.128\).

  1. Cumulative Distribution Function (CDF):

\[ P(X \leq k) = 1 - (1-p)^k \]

This gives the probability that success occurs on or before trial \(k\).

  1. Statistical Measures:

  • Mean (Expected Value):

\[ E[X] = \frac{1}{p} \]

(average number of trials until first success)

  • Variance:

\[ Var(X) = \frac{1-p}{p^2} \]

  • Mode: Always at \(k=1\) (success happens on the very first trial is the most likely outcome).

üéØ Applications#

  • Quality control: Number of items checked until the first defective product appears.

  • Marketing: Number of calls until the first lead converts.

  • Reliability engineering: Time until the first system failure.

‚úÖ Key Insight: The geometric distribution is all about ‚Äúwaiting time until first success‚Äù, unlike the binomial distribution, which counts the total number of successes in a fixed number of trials.

import numpy as np
import matplotlib.pyplot as plt
import seaborn as sns
from scipy.stats import geom

# Parameters
p = 0.3   # probability of success
k_values = np.arange(1, 15)  # number of trials

# PMF and CDF
pmf_values = geom.pmf(k_values, p)
cdf_values = geom.cdf(k_values, p)

# Plot PMF
plt.figure(figsize=(12, 5))

plt.subplot(1, 2, 1)
sns.barplot(x=k_values, y=pmf_values, color="skyblue", edgecolor="black")
plt.title("Geometric Distribution - PMF (p=0.3)")
plt.xlabel("Trial (k)")
plt.ylabel("P(X=k)")

# Plot CDF
plt.subplot(1, 2, 2)
sns.lineplot(x=k_values, y=cdf_values, marker="o", color="red")
plt.title("Geometric Distribution - CDF (p=0.3)")
plt.xlabel("Trial (k)")
plt.ylabel("P(X ‚â§ k)")

plt.tight_layout()
plt.show()
