The Poisson Distribution (Discrete)#

The Poisson distribution is a discrete probability distribution that models the number of times an event occurs in a fixed interval of time, space, distance, or area, when events occur independently and at a constant average rate.

Probability Mass Function (PMF)#

For a Poisson random variable \(X \sim Poisson(\lambda)\):

\[ P(X = k) = \frac{e^{-\lambda} \lambda^k}{k!}, \quad k = 0, 1, 2, \dots \]

Where:

  • \(k\) = number of occurrences (discrete values)

  • \(\lambda > 0\) = average rate of events per interval

  • \(e \approx 2.718\) (Euler‚Äôs constant)

Parameters#

  • \(\lambda\) (lambda) = expected number of events per interval

  • Distribution is completely defined by \(\lambda\).

Statistical Measures#

  • Mean (Expected Value):

\[ E[X] = \lambda \]

  • Variance:

\[ Var[X] = \lambda \]

  • Standard Deviation:

\[ \sigma = \sqrt{\lambda} \]

  • Skewness:

\[ \text{Skewness} = \frac{1}{\sqrt{\lambda}} \]

  • Excess Kurtosis:

\[ \text{Kurtosis} = \frac{1}{\lambda} \]

Characteristics#

  1. Discrete: takes integer values \(k = 0,1,2,\dots\)

  2. Equal mean and variance (\(E[X] = Var[X] = \lambda\))

  3. Shape depends on \(\lambda\):

    • Small \(\lambda\): distribution skewed right (most mass near 0).

    • Large \(\lambda\): approaches Normal distribution.

  4. Additivity: If \(X_1 \sim Poisson(\lambda_1)\) and \(X_2 \sim Poisson(\lambda_2)\), then \(X_1 + X_2 \sim Poisson(\lambda_1 + \lambda_2)\).

Examples#

  1. Call center problem: If a call center receives on average \(\lambda = 3\) calls per minute, what is the probability of receiving exactly 5 calls in a minute?

    \[ P(X=5) = \frac{e^{-3} 3^5}{5!} = \frac{e^{-3} \cdot 243}{120} \approx 0.1008 \]

    ‚úÖ About 10% chance of getting exactly 5 calls.

  1. Website traffic: Suppose a website receives on average \(\lambda = 10\) visits per hour.

    • Probability of no visits in an hour:

    \[ P(X=0) = \frac{e^{-10} 10^0}{0!} = e^{-10} \approx 0.000045 \]

    ‚úÖ Almost impossible to have zero visits.

Applications#

  • Event counts in time/space (calls, arrivals, clicks, earthquakes, defects, etc.)

  • Reliability engineering (number of failures per unit time).

  • Biostatistics (disease cases in a region).

  • Queueing theory (customers arriving in a queue).

‚úÖ In short: The Poisson distribution is a discrete distribution where the probability of observing \(k\) events depends only on the average rate \(\lambda\). Its key property is mean = variance = \(\lambda\).

import numpy as np
import matplotlib.pyplot as plt
import seaborn as sns
from scipy.stats import poisson

# Parameters
lambdas = [1, 4, 10]   # Different Œª values to demonstrate
x = np.arange(0, 20)   # Range of possible counts

plt.figure(figsize=(12, 8))

# Plot PMFs for different Œª
for i, lam in enumerate(lambdas, 1):
    plt.subplot(2, 3, i)
    pmf = poisson.pmf(x, lam)
    sns.barplot(x=x, y=pmf, color="skyblue", edgecolor="black")
    plt.title(f"Poisson PMF (Œª = {lam})")
    plt.xlabel("Number of events (k)")
    plt.ylabel("P(X = k)")

# Simulated samples for visualization
for i, lam in enumerate(lambdas, 1):
    plt.subplot(2, 3, i+3)
    samples = np.random.poisson(lam, 1000)
    sns.histplot(samples, bins=20, kde=True, color="lightcoral", edgecolor="black")
    plt.title(f"Simulated Data (Œª = {lam})")
    plt.xlabel("Observed counts")
    plt.ylabel("Frequency")

plt.tight_layout()
plt.show()
